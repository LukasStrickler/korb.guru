# Convex (@korb/convex)

Realtime backend for Korb — collaborative state, reactive queries, and mutations. Use this app so the mobile client can subscribe to live data and run mutations; schema and functions are in `convex/` (TypeScript).

- [Setup](#setup) · [Important commands](#important-commands) · [Project structure](#project-structure) · [Docs](#docs)

## Setup

1. **Log in to Convex** (one-time):

   ```bash
   npx convex login
   ```

   Install workspace deps from repo root: `pnpm install`.

2. **Set environment variables** — Create `apps/convex/.env` (or use root `.env`):

   ```env
   CONVEX_DEPLOYMENT=<your-dev-deployment-url>
   ```

   Set the **mobile** env `EXPO_PUBLIC_CONVEX_URL` to the same deployment. See [Local Development](../../.docs/guides/local-dev.md).

3. **Run Convex dev** (from repo root or this directory):
   ```bash
   pnpm --filter @korb/convex dev
   ```
   This syncs schema and functions and gives you a deployment URL for `EXPO_PUBLIC_CONVEX_URL` and `CONVEX_DEPLOYMENT`.

## Important commands

Run from **repo root** with `pnpm --filter @korb/convex <script>`, or from `apps/convex` after setup.

| Command                                | Description                                 |
| -------------------------------------- | ------------------------------------------- |
| `pnpm --filter @korb/convex dev`       | Run Convex dev (sync schema and functions). |
| `pnpm --filter @korb/convex deploy`    | Deploy to production.                       |
| `pnpm --filter @korb/convex lint`      | Run ESLint.                                 |
| `pnpm --filter @korb/convex typecheck` | TypeScript check.                           |
| `pnpm --filter @korb/convex test`      | Run Vitest tests.                           |

From inside `apps/convex`: use `pnpm dev`, `pnpm lint`, `pnpm typecheck`, `pnpm test` (same script names).

## Project structure

| Path                    | Description                                                               |
| ----------------------- | ------------------------------------------------------------------------- |
| `convex/schema.ts`      | Convex schema (tables, indexes).                                          |
| `convex/*.ts`           | Queries, mutations, actions (e.g. `users.ts`, `recipes.ts`, `handle.ts`). |
| `convex/_generated/`    | Generated by Convex (do not edit).                                        |
| `convex/auth.config.ts` | Convex auth (e.g. Clerk).                                                 |
