<Title>Korb Guru – Mobile components</Title>

Component library for the Korb Guru app (Expo React Native). Use the sidebar to browse components. Default viewport is mobile (414×896); use the toolbar to change size or zoom.

## Run Storybook

```sh
pnpm storybook:web   # browser at http://localhost:6006
pnpm storybook       # Metro, then i / a / w for iOS / Android / web
```

See `.docs/guides/storybook-mobile.md` in the repo for portable stories and Jest.
